Star Wars Social Network

On this website, you can interactively explore interactions from all Star Wars episodes. By clicking on a node, you can view detailed information. Additionally, you can select different episodes using the dropdown menu to tailor your exploration. With help of the range sliders, you also have the capability to set a threshold for the minimum number of interactions and for scene apperances, allowing for a more customized viewing experience.
